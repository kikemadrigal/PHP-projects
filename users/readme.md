# Login-user

API Rest que te proporcina una autenticación de usuarios para tus webs.

Los usuarios son kike contraseña 1234, juan contraseña 1234, pepe contraseña 1234.

# Instalation

1. Descarga XAMPP: https://www.apachefriends.org/es/index.html

2. Abre XAMPP control panel e inicia PHP y mysql.

3. Ve a http://localhost/phpmyadmin/ para abrir phpmyadmin y crea una nueva base de datos llamada usuarios.

4. importa el archivo schema.sql para que cree la tabla e inserte los usuarios.

5. Copia y pega la carpeta de estre proyecto dentro de C:\xampp\htdocs

6. Listo, En tu navegador web ve a http://localhost/users/
